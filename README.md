# 🚀 Machine-Learning-Analysis-of-Amazon-Reviews 

This project explores customer behavior using the Amazon Reviews dataset through data preprocessing, feature engineering, and unsupervised machine learning techniques.

---

## 📊 Project Overview

- Performed Exploratory Data Analysis (EDA)
- Applied data preprocessing (missing values, outliers, encoding, scaling)
- Engineered features such as review length
- Applied PCA for dimensionality reduction
- Implemented clustering models:
  - KMeans
  - DBSCAN
- Evaluated models using:
  - Silhouette Score
  - Davies-Bouldin Index
  - Calinski-Harabasz Index

---

## 🤖 Key Findings

KMeans produced more stable and interpretable clusters compared to DBSCAN, indicating structured customer behavior patterns.
